<h1 align="center">pith-ui-themes</h1>

Radix Themes is a pre-styled component library that is designed to work out of the box with minimal configuration.

[Pith UI](https://github.com/pith-ui/pith-ui) is a Rust port of [Radix](https://www.radix-ui.com/).

## Documentation

See [the Pith UI book](https://pith-ui.dev/) for documentation.
